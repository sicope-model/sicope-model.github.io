Bug

Bug is created automatically while running task. Bug also is reduced automatically. Bug can also is reduced manually later.

Bug#

Bug contains useful information for tester to verify:

  • Message
  • Steps
  • Recorded Video

We can also open/close the bug. But it's just a simple flag. Don't use it to manage the bug. This tool is not a bug tracker.
